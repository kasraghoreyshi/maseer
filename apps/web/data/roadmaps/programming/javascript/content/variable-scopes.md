## محدوده متغیر ها

در جاوا اسکریپت، متغیر ها محدوده مشخصی دارند. به عنوان مثال، از یک متغیری که در یک تابع تعریف شده است، نمی توانید استفاده کنید مگر آنکه استفاده تان درون خود آن تابع باشد.
