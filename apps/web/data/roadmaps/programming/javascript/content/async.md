## جاوا اسکریپت ناهمگام یا Asynchronous Javascript

جاوا اسکریپت ناهمگام یک مبحث نسباتا پیچیده در یادگیری جاوا اسکریپت است که بیان می کند برنامه شما قادر باشد یک وظیفه طولانی را انجام دهد درحالی که باقی ایونت ها، همچنان قابل اجرا باشند.
