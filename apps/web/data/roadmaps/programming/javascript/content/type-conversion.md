## تبدیل تایپ ها به یکدیگر

تبدیل تایپ یعنی تبدیل یک نوع داده به نوعی دیگری از داده. تبدیل های ضمنی مواقعی اتفاق می افتد که زبان مورد نظر، به صورت خودکار تایپ را تبدیل می کند به تایپی دیگر.
