## Components یا کامپوننت ها

کامپوننت ها جزو یکی از مباحث اصلی ری اکت و فریمورک های مشابه می باشد. به طور خلاصه، کامپوننت ها تکه کد های مختلف هستند که قابل استفاده مجدد در سراسر سایت شما هستند.
