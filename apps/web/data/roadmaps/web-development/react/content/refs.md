## رفرنس ها یا Refs

Ref در ری اکت، یک خاصیت است که امکان این را به شما می دهد که یک node در DOM را آدرس دهید تا بتوانید روی آن، کار هایی که با Javascript DOM می توانید انجام بدید را انجام دهید.
